Documenting my computer science journey through written tutorials and concept breakdowns to build my technical communication skills by applying the Feynman Technique. Preparing for rigorous upper-division coursework at UC Irvine. 
🎓 UCI Computer Science Prep Notes

Welcome to my active learning repository! This project serves as my digital brain and teaching portfolio as I prepare for upper-division computer science coursework at the University of California, Irvine.

🌟 The Mission

To master complex computer science concepts, you have to be able to teach them. I am using the Feynman Technique to deconstruct rigorous academic topics and rewrite them as accessible, easy-to-understand tutorials.

My ultimate goal is to become a world-class software engineer and educator, providing equitable access to technical information for everyone.

📚 Current Study Tracks

This repository is organized by the core university subjects I am currently mastering:

/Discrete-Math: Predicate logic, proofs, and graph theory (Prep for I&C SCI 6D).

/MIPS-Architecture: Low-level hardware-software interface and assembly (Prep for I&C SCI 51/53).

/Data-Structures: Deep dives into C++ implementations of Trees, Graphs, and Hash Maps (Prep for I&C SCI 46).

/Statistics: Calculus-based probability and statistics (Prep for STATS 67).

💡 How I Use This Repo

Every time I encounter a difficult concept, I don't just solve it—I explain why it works. You will find:

Markdown Tutorials: Step-by-step breakdowns of complex algorithms and mathematical proofs.

Code Snippets: Heavily commented C++, Python, and MIPS Assembly code.

Concept Maps: Visual representations of how different systems connect.

“If you can't explain it simply, you don't understand it well enough.”
